**Taroudant** is a walled city in Morocco's Souss Valley, nestled between the High Atlas and Anti-Atlas mountains. Its iconic **ochre ramparts**, stretching over **7.5 kilometers**, date back to the **16th century Saadian dynasty**. The city is home to mosques, fountains, souks, and gates ‚ÄĒ all classified as irreplaceable cultural heritage.

Yet these structures are **aging**. Cracks form. Walls erode. Foundations weaken. And until now, there has been **no automated system** to detect, score, and alert authorities about the structural degradation of these monuments.

**That is the problem Taroudant Heritage Shield solves.**

## ūüí° The Vision

```
Imagine a world where a crack in Bab El Khemis is detected on Monday,
scored automatically by Tuesday, and a restoration order is signed by Wednesday.

That is the world this platform builds.
```

Taroudant Heritage Shield is a **full-stack web application** that:

- ūüďć **Catalogs** every monument and rampart of Taroudant with GPS coordinates, photos, and historical data
- ūüĒć **Tracks** structural inspections performed by certified field experts
- ūüßģ **Scores** vulnerability automatically using age + crack severity formulas via MySQL stored procedures
- ūüö® **Alerts** municipal authorities in real-time when critical risk is detected via database triggers
- ūüďĄ **Generates** encrypted expert reports accessible only to authorized decision-makers
- ūüĆź **Displays** a public monument health catalogue for citizens and researchers

## ūüßĪ SQL Business Logic

### ‚öôÔłŹ Stored Procedure 1 ‚ÄĒ `CalculateVulnerabilityScore`

```sql
-- Automatically called by Trigger 1 after each crack is logged
-- Weighted formula: crack severity √ó age multiplier √ó category factor
-- Step 1 ‚ÄĒ Crack severity weights (length-adjusted)
--   minor:    1  √ó min(length_cm / 10,  2)
--   moderate: 3  √ó min(length_cm / 20,  3)
--   major:    7  √ó min(length_cm / 30,  4)
--   critical: 15 (always maximum, length irrelevant)
-- Step 2 ‚ÄĒ Age multiplier: 1.0 + min(age_years / 1000, 0.5)
--   ‚Üí 1.0 for new structures, up to 1.5 for 500+ year old ones
--   ‚Üí applied only when cracks exist (old monument with no cracks = LOW)
-- Step 3 ‚ÄĒ Age score: min(age_years / 20, 10)  [capped at 10]
-- Step 4 ‚ÄĒ Category durability factor
--   Rempart 0.8 | Porte de ville 0.9 | Kasbah 0.85
--   Mosqu√©e 1.0 | Maison traditionnelle 1.2 | Jardin 1.1
-- Step 5 ‚ÄĒ Final score
--   crack_score  = raw_crack_score √ó age_multiplier
--   total_score  = FLOOR((crack_score + age_score) √ó category_factor)
-- Override: any critical crack forces minimum HIGH risk
CALL CalculateVulnerabilityScore(inspection_id);
-- Output: INSERT into vulnerability_scores
```

| Score Range | Risk Level |
|---|---|
| 0 ‚Äď 15 | ūüüĘ LOW |
| 16 ‚Äď 35 | ūüü° MEDIUM |
| 36 ‚Äď 60 | ūüü† HIGH |
| 61+ | ūüĒī CRITICAL |
