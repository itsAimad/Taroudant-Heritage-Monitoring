**Taroudant** is a walled city in Morocco's Souss Valley, nestled between the High Atlas and Anti-Atlas mountains. Its iconic **ochre ramparts**, stretching over **7.5 kilometers**, date back to the **16th century Saadian dynasty**. The city is home to mosques, fountains, souks, and gates вҖ” all classified as irreplaceable cultural heritage.

Yet these structures are **aging**. Cracks form. Walls erode. Foundations weaken. And until now, there has been **no automated system** to detect, score, and alert authorities about the structural degradation of these monuments.

**That is the problem Taroudant Heritage Shield solves.**

---

## рҹ’Ў The Vision

```
Imagine a world where a crack in Bab El Khemis is detected on Monday,
scored automatically by Tuesday, and a restoration order is signed by Wednesday.

That is the world this platform builds.
```

Taroudant Heritage Shield is a **full-stack web application** that:

- рҹ“Қ **Catalogs** every monument and rampart of Taroudant with GPS coordinates, photos, and historical data
- рҹ”Қ **Tracks** structural inspections performed by certified field experts
- рҹ§® **Scores** vulnerability automatically using age + crack severity formulas via MySQL stored procedures
- рҹҡЁ **Alerts** municipal authorities in real-time when critical risk is detected via database triggers
- рҹ“„ **Generates** encrypted expert reports accessible only to authorized decision-makers
- рҹҢҗ **Displays** a public monument health catalogue for citizens and researchers

| Threat | Protection |
|---|---|

| рҹ”“ Password theft | bcrypt hashing вҖ” plain text never stored |
| рҹ‘ҒпёҸ Untracked access | Every sensitive action logged in audit_logs |

## рҹ§ұ SQL Business Logic

### вҡҷпёҸ Stored Procedure 1 вҖ” `CalculateVulnerabilityScore`

```sql
-- Automatically called by Trigger 1 after each crack is logged
-- Computes risk level from monument age + crack severity weights
CALL CalculateVulnerabilityScore(inspection_id);
-- Output: INSERT into vulnerability_scores
```

### рҹ“„ Stored Procedure 2 вҖ” `GenerateMonumentReport`

```sql
-- Called by inspector when field work is complete
-- Compiles all inspection data вҶ’ encrypts вҶ’ saves report
CALL GenerateMonumentReport(monument_id, inspection_id, generated_by);
-- Output: INSERT into reports (encrypted content)
```

### вҡЎ Trigger Summary

| Trigger | Fires On | Action |
|---|---|---|
| `after_crack_insert` | INSERT on `cracks` | Calls `CalculateVulnerabilityScore` |
| `after_score_insert` | INSERT on `vulnerability_scores` | If HIGH/CRITICAL вҶ’ INSERT notifications |
| `after_report_insert` | INSERT on `reports` | INSERT into `audit_logs` |
