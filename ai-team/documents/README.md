
# 🏛️ Taroudant Heritage Shield

Academic project comparing AI-assisted vs traditional 
development of a heritage monument monitoring system 
for the city of Taroudant, Morocco.

## Project Structure

| Folder | Description |
|---|---|
| ai-team/ | Version built with AI assistance (Claude + Cursor) |
| team-without-ai/ | Version built with traditional methods |

## What the system does
- Monitors historic monuments and ramparts of Taroudant
- Automatically scores structural vulnerability from field inspections
- Alerts municipal authorities when critical risk is detected
- Provides encrypted expert reports for decision-makers
- Shows public monument health catalogue for citizens

## Tech Stack (AI Team)
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MySQL (3NF, stored procedures, triggers)
- Security: JWT, RBAC, AES-256, bcrypt
